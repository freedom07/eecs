# 목차

- Chapter 2. Intelligent Agents(지능적 에이전트)

- Chapter 7. 논리적 에이전트
- Chapter 8. 1차 논리
- Chapter 9. 1차 논리의 추론
- Chapter 10. 지식 표현
- Chapter 11. 자동 계획 수립

- Chapter 15. 확률적 프로그래밍
- Chapter 16. 간단한 의사결정
- Chapter 17. 복잡한 의사결정
- Chapter 18. 다중 에어진트 의사결정

PART 5 기계학습 (2판 ~)
- Chapter 19. Learning from Examples(견본에서 배우는 학습)
